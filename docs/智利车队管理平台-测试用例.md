# 智利车队管理平台 - 测试用例

| 模块 | 用例 ID | 功能点 | 测试步骤 | 预期结果 | 优先级 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **登录** | TC-LG-01 | 用户登录 | 输入账号密码，点击登录 | 成功跳转至首页看板 | P0 |
| **多语言** | TC-I18N-01 | 语言切换 | 在 Topbar 点击切换语言按钮 | 全站文本、日期格式、Antd 内部组件同步切换 | P0 |
| **多语言** | TC-I18N-02 | 刷新状态保持 | 切换为西班牙语后刷新页面 | 页面依然显示为西班牙语，且无英文闪烁 | P1 |
| **首页** | TC-DB-01 | 统计指标 | 查看统计卡片数据 | 显示车辆总数、在线/离线、今日里程及预警数 | P1 |
| **车辆管理** | TC-VH-01 | 车辆搜索 | 输入 VIN 或车牌号点击查询 | 表格仅显示符合条件的车辆 | P1 |
| **车辆管理** | TC-VH-02 | 车辆详情 | 点击列表操作栏“详情” | 进入详情页，显示车辆信息、电池监控、行程记录等 Tab，不应白屏 | P0 |
| **实时监控** | TC-MT-01 | 实时位置 | 点击二级菜单“实时位置” | 地图显示全量车辆 Marker，点击 Marker 显示提示框 | P1 |
| **实时监控** | TC-MT-02 | 轨迹回放 | 选择车辆及时间范围，点击播放 | 地图绘制轨迹路径，Marker 沿路径移动 | P2 |
| **风险预警** | TC-RK-01 | 围栏预警 | 切换至围栏预警 Tab | 列表显示触发围栏报警的车辆、围栏名称、时间 | P1 |
| **围栏管理** | TC-FN-01 | 查看围栏详情 | 点击列表中的“查看(View)” | 弹出 Modal，并在 Modal 中加载地图显示围栏圆形范围 | P0 |
| **围栏管理** | TC-FN-02 | 新建围栏 | 点击新建，输入名称、地址、半径并提交 | 列表出现新围栏记录 | P1 |
| **租户管理** | TC-TN-01 | 列表分页 | 查看租户管理列表底部 | 显示总记录数（应为 24 条），点击分页按钮切换数据 | P0 |
| **租户管理** | TC-TN-02 | 账号下发 | 为租户下发管理员账号 | 弹出确认框显示自动生成的账号及随机密码 | P2 |
| **菜单导航** | TC-NAV-01 | 层级点击 | 点击左侧“运营管理”菜单 | 一级菜单展开显示子项，不可直接跳转；点击子项正常跳转 | P0 |
| **菜单导航** | TC-NAV-02 | Tab 联动 | 点击“实时监控”下的“轨迹回放” | 页面跳转至 Monitor，且自动切换到“轨迹回放”Tab | P1 |
| **UI 表现** | TC-UI-01 | 字号检查 | 检查全站表格及标题字号 | 视觉上无拥挤，字号较原版本略大（15px），清晰易读 | P2 |
