# Review_05 改动 · 测试计划与用例

## 测试范围

本次改动涉及 4 个方面，测试覆盖每个改动的正确性：

| 改动 | 风险点 |
|---|---|
| Sidebar 子菜单 i18n 化（19 个新键） | 键名拼写错误、切换语言后不生效 |
| Auth guard 清理 | 登录/未登录路由跳转异常 |
| dayjs locale 补全 | dayjs 本地化失效 |
| 导入路径 `../../` → `@/` | 路径解析失败导致构建/运行时崩溃 |

## 测试环境

- 构建: `npx vite build`
- 运行: `npx vite preview` → `http://localhost:3000`
- 工具: 浏览器预览 + 控制台
