# 智利车队管理平台 -- UI字段梳理报告（极其详尽版）

**基于文档**: PRD V1.0 | **覆盖范围**: 全部功能模块的所有页面、弹窗、菜单、按钮、筛选项、列表、Tab、图表

---

## 一、全局逻辑

### 1.1 时间处理
- 时区：智利时区(Santiago)
- 冬令时：UTC-4（每年4月第一个星期六24:00 至 9月第一个星期六24:00）
- 夏令时：UTC-3（每年9月第一个星期六24:00 至 次年4月第一个星期六24:00）
- 通用格式：`YYYY-MM-DD HH:mm:ss`

### 1.2 VIN码脱敏
- 规则：17位VIN码，保留前6位+后4位，中间7位用`*`替换
- 示例：`LSG4AA*******5678`
- 覆盖：全局所有前端展示界面
- 例外：数据导出文件(Excel)、数据导入文件使用明文

### 1.3 车牌号脱敏
- 长度≤4位：不脱敏
- 长度≥5位：首尾各保留2位，中间用`*`替换
- 示例：`ABCD12`→`AB**12`；`ABCDE`→`AB*DE`

### 1.4 地理位置精度
- 最高仅保留至街道/路口级别

### 1.5 检索匹配
- 支持明文和脱敏字符模糊匹配，数据库存储明文，前端执行脱敏渲染

### 模块2：基础能力

#### 修改密码弹窗
- 触发1：新用户/重置密码未修改过，登录后自动弹出
- 触发2：右上角用户昵称下拉→修改密码
- 字段：当前密码(必填)、新密码(8~18位,数字+英文+特殊字符)、确认新密码(与新密码一致)
- 按钮：`确定`、`取消`
- 提示：当前密码错→`当前密码错误`；格式不符→`请输入输入8~18位字符，支持数字、英文、特殊字符`；新旧一致→`新密码不能与当前密码相同`；确认不一致→`新密码不一致`

#### 语言切换
- 中文、英文(默认)、西班牙文

#### 租户切换
- 平铺展示，最新在最下面；单租户隐藏；多租户展示切换按钮
- 到期租户显示`服务过期`；记忆最近一次选择

### 模块4：业务管理

#### 4.1 租户权限
- 根级租户(苇渡汽车)：不可编辑/删除
- 一级租户：不可编辑/删除，可添加一个下级(添加后入口隐藏)
- 二级及以下：编辑名称、删除(有下级不可删除+提示)、新增下级(仅一个)
- 功能权限：根级默认不可配；一级租户维度配置；二级及以下层级维度配置

#### 4.2 租户信息
- 查询：模糊查租户名称
- 树级结构展示，除根级外可查看详情
- 仅面向应用管理员

#### 4.3 资产划拨
**列表**：序号、VIN码(脱敏)、归属租户、同步时间 | 操作：资产划拨/划拨记录/删除
**排序**：创建时间倒序 | **分页**：默认10，可选10/20/50/100
**筛选**：VIN码(精准)、归属租户(多选,含"未划拨")、同步时间(日期区间)
**按钮**：同步数据、批量资产划拨(未勾选→`请选择指定数据操作`)

**批量划拨弹窗**：归属租户(租户树,必选) | 按钮：确定/取消
**单条划拨弹窗**：VIN码(仅展示)、归属租户(已划拨回显/未划拨→`请选择归属数据`) | 按钮：确定/取消
**划拨记录**：划拨时间(倒序)、变更人、划拨前租户、划拨后租户
**删除**：已关联设备→`该资产已关联设备，不可删除`；未关联→二次确认

#### 4.4 用户管理(业务管理)
**布局**：左侧租户树(默认根级) + 右侧列表
**列表**：序号、用户昵称、登录邮箱、角色(多个"、"分隔)、创建时间 | 操作：重置密码/编辑/删除
**排序**：创建时间倒序 | **分页**：默认10，可选10/20/50/100
**筛选**：用户昵称(模糊)、登录邮箱(模糊)、角色(多选下拉+模糊)、创建时间(日期区间)

**新增用户第一步**：用户昵称(≤20字符,必填)、登录邮箱(必填,格式校验→`请输入正确格式的邮箱`)、角色(多选必选,管理员角色置灰) | 按钮：下一步/返回
**第二步(不存在)**：用户ID、登录邮箱、初始密码(大写英文+数字8位) | 按钮：复制信息(toast:`复制成功`)/完成
**第二步(已存在)**：用户ID、登录邮箱、用户昵称 | 按钮：复制信息/使用该账号创建/上一步/完成

**重置密码**：二次确认→展示用户昵称、登录邮箱、新密码 | 按钮：复制信息/关闭
**编辑弹窗**：用户昵称、登录邮箱、角色(管理员置灰) | 邮箱已用→飘红
**删除**：管理员不可删；普通→二次确认

#### 4.5 角色管理(业务管理)
**布局**：左侧租户树 + 中间角色列表 + 右侧权限面板
**排序**：创建时间正序(最新最下) | 管理员角色：不可编辑/删除/修改权限
**新增**：角色名称(≤10字符,唯一→重复飘红`角色已存在`)、功能权限(编辑状态)、数据权限(默认全部下级)
**删除**：未关联→`删除后数据无法恢复，是否继续？`；已关联→`该角色已关联用户，删除后用户无相关权限，是否继续？`

**Tab2 驾驶预警记录**：预警名称(6项枚举)、车速(km/h)、预警时间
> 对车一级预报警、对车二级紧急报警、对车AEB制动、对人一级预警、对人二级预警、对人AEB制动

**Tab3 电池监控信息**：SOC(%)、电池健康度(%)、电池温度(℃)、续航里程(km)

**Tab4 充电记录**：电压(V)、电流(A)、功率(kW)、充电前电量(%)、充电后电量(%)、充电时长、累计充电次数、记录时间

**Tab5 行程记录**：开始时间、结束时间、起点、终点、行驶里程、行程时长、平均车速、预警次数(驾驶+围栏+故障+低电量)

**Tab6 维修记录**：维修类型(故障类/电池类)、维修描述、开始时间、结束时间、操作人、维修状态(维修中/维修完成)
按钮：`完成维修`(仅维修中显示)

**Tab7 里程报表**：折线图 | X轴：日(近30天)/周(近30周)/月(近12月)/年(近3年) | Y轴km | 选项：日/周/月/年

### 模块20：系统管理

#### 用户管理
与业务管理下用户管理一致，**无左侧租户树**，根据登录用户租户层级查询。

#### 角色管理
与业务管理下角色管理一致，根据登录用户租户层级查询。

#### 租户管理员同步规则
租户创建→同步管理员角色 | 管理员账号→角色固定不可选 | 设置后→同步到用户管理 | 管理员角色不可删/不可配/仅查看 | 管理员账号不可删/角色不可改(置灰) | 新增用户不可选管理员角色(置灰)

#### 日志审计
| 列名 | 格式 |
|------|------|
| 序号 | 序列号 |
| 操作时间 | YYYY-MM-DD HH:mm:ss (Santiago) |
| 操作人昵称 | 用户昵称 |
| 操作账号 | 登录邮箱 |
| 所属租户 | 租户名称 |
| 操作IP | IPv4/IPv6 |
| 操作类型 | `一级模块-二级模块` |
| 事件描述 | 拼装描述(含脱敏) |
| 操作结果 | 成功/失败(失败Hover显示原因) |

保留：至少180天
**筛选**：操作账号(≤50字符,模糊)、操作类型(多选下拉)、操作时间范围(跨度≤180天,仅过去180天)、操作结果(全部/成功/失败)

**采集范围**：安全账户(登录/退出/改密/重置) | 租户权限(CRUD租户/用户/角色/权限) | 资产车辆(同步/划拨/删除/导入) | 围栏(CRUD/开关/车辆) | 维修(新建/报修/完成/删除) | 数据(导出/下载)
