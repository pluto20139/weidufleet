01

登录及基础能力

▼
模块概述

对接用户中心，提供平台登录认证、密码管理、多语言切换及租户切换等基础能力。所有功能均复用基线能力，但需适配智利车队的用户体系与租户关系。

功能结构
功能类型说明
项目应用登录基线调用公共登录接口，支持账号+密码+验证码
修改密码基线首次登录强制修改；用户主动修改。密码规则：8~18位，数字+英文+特殊字符
语言切换基线汉语 / 英语 / 西班牙语（智利市场）
租户切换基线根据用户关联的租户列表平铺展示，无层级
业务逻辑
场景逻辑
首次登录用户未修改过密码 → 自动弹出修改密码弹窗 → 修改后方可进入首页
租户切换仅关联1个租户 → 隐藏切换按钮。关联多个 → 展示按钮，点击展示租户列表；记录最近一次选择的租户，下次登录默认选中
语言切换默认跟随系统/浏览器语言
支持汉语、英语、西班牙语
切换后全局生效，持久化到用户配置
服务到期租户关联的一级租户服务到期 → 该一、二、三级租户均显示"服务过期"
异常提示
场景提示文案
账号密码不匹配 / 账号不存在"账号密码错误"
验证码不正确"验证码错误"
当前密码错误（修改密码）"当前密码错误"
新密码格式不符"请输入8~18位字符，支持数字、英文、特殊字符"
新密码与当前密码一致"新密码不能与当前密码相同"
确认新密码不一致"新密码不一致"
API 接口
POST /api/v1/auth/login — 用户登录
PUT /api/v1/auth/password — 修改密码
GET /api/v1/user/tenants — 获取用户关联租户列表
PUT /api/v1/user/switch-tenant — 切换当前租户
GET /api/v1/user/language — 获取语言偏好
PUT /api/v1/user/language — 设置语言偏好
UI 组件拆解
组件说明
登录页账号输入 + 密码输入 + 验证码 + 登录按钮
修改密码弹窗当前密码 / 新密码 / 确认新密码 + 确定/取消
语言切换下拉右上角用户昵称下拉菜单，三个语言选项
租户切换面板点击头像/昵称后展示租户列表，支持"服务过期"标记
测试要点
  • 首次登录强制修改密码流程
  • 租户切换后菜单权限刷新
  • 三种语言切换后文案完整
  • 密码规则校验（8~18位含特殊字符）
  • 服务到期租户标记与进入限制
  • 登录失败3次后验证码机制
02

首页看板

▼
模块概述

展示车队运营核心指标的仪表盘，包括车辆状态统计、驾驶风险分布图表、车辆实时位置地图和今日预警排行榜。帮助管理者快速掌握车队整体运营状况。

功能结构
区域内容数据来源
数据指标车辆总数 / 在线数 / 离线数 / 今日里程 / 累计总里程 / 今日驾驶预警 / 今日围栏预警 / 今日低电报警OBD + 车身
数据图表驾驶风险分布柱状图（急加速/急减速/急转弯/疲劳驾驶/AEB），支持今日/近7日/近30日切换OBD
实时位置地图以车队运营城市为中心的地图，展示所有车辆实时位置、车辆信息、电池信息、速度OBD
今日预警排行榜按车辆展示今日驾驶预警/围栏预警/故障预警/低电报警次数，按总计降序OBD + 车身
数据模型
字段类型计算规则
车辆总数int当前租户及下级租户名下所有车辆总数
在线/离线数int依据OBD设备在线/离线状态统计
今日里程float(km)所有车辆今日行驶里程总和
驾驶风险分布ratio每类风险事件数量 / 总事件数 × 100%
预警排行榜list按总计降序，展示车牌+4类预警次数+总计
API 接口
GET /api/v1/dashboard/overview — 获取看板概览数据（指标+排行榜）
GET /api/v1/dashboard/driving-risk?range=today|7d|30d — 驾驶风险分布
GET /api/v1/dashboard/real-time-map — 实时位置地图数据
UI 组件拆解
组件说明
统计指标卡8个指标卡片，2×4 或 4×2 网格布局
柱状图ECharts/Chart.js，支持时间范围切换（今日/近7日/近30日）
实时位置地图Mapbox/高德地图，车辆图标打点，聚合展示，支持缩放/拖拽
预警排行榜列表组件，带排名序号和颜色标记
测试要点
  • 指标数据与后端统计一致
  • 图表时间范围切换刷新数据
  • 地图车辆位置定时刷新
  • 排行榜排序正确性
  • 空数据状态展示
03

业务管理

▼
模块概述

面向应用管理员（根级租户）的多层级租户管理体系，涵盖租户层级定义、功能权限配置、资产划拨、用户与角色管理。

3.1 租户权限
功能逻辑
租户层级管理根级（苇渡汽车）→ 一级租户 → 二级及以下（同层级共享权限模板）
层级增删改查根级初始化，不可编辑。一级从租户管理同步，不可编辑/删除。二级支持编辑名称、删除（无下级时）、新增下级。每级仅可添加一个直接下级
功能权限配置根级：不可配置（产品定义）。一级：按租户维配置全部功能权限。二级及以下：按层级模板批量配置
3.2 租户信息
功能逻辑
租户树按树级结构展示全部租户，支持名称模糊搜索
租户详情点击任意非根级租户节点，展示基础信息 + 管理员账号信息
权限范围仅应用管理员（根级租户管理员）可访问此页面
3.3 资产划拨
功能逻辑
资产列表展示应用下全部车辆资产（IOT平台按渠道标识同步落库），字段：VIN码、归属租户、同步时间、操作（划拨/记录/删除）
同步数据首次全量同步，后续增量同步（以VIN为唯一标识）
资产划拨单条或批量选择资产 → 选择目标租户 → 更新归属。已划拨的资产再次划拨时覆盖原租户
划拨记录展示归属租户变更历史：划拨时间、变更人、划拨前/后租户
删除资产已关联设备的资产不可删除；未关联的二次确认后删除
筛选VIN码（精准）、归属租户（级联选择，含"未划拨"）、同步时间（日期区间）
3.4 用户管理
功能逻辑
用户列表根据左侧选中租户查询该租户下用户列表。字段：序号/用户昵称/登录邮箱/角色/创建时间/操作
新增用户（两步）Step1: 填写昵称、邮箱、角色。Step2: 检查用户中心是否存在该邮箱→不存在则创建用户+分配角色；存在则关联已有用户+追加角色
编辑用户可编辑昵称、邮箱、角色（管理员角色置灰不可改）。邮箱唯一性校验
重置密码二次确认后重置为8位随机密码（大写字母+数字），展示新密码并支持复制
删除用户仅删除该用户在当前租户下的角色权限关系。管理员角色账号不可删除
筛选用户昵称（模糊）、登录邮箱（模糊）、角色（多选）、创建时间（日期区间）
3.5 角色管理
功能逻辑
角色列表根据左侧选中租户查询角色信息，按创建时间正序。管理员角色（系统自动创建）不可编辑/删除
新增角色输入角色名称（该校验唯一性）→ 配置功能权限（可选范围=该租户层级可配置范围）→ 数据权限默认全部下级租户
删除角色未关联用户：二次确认后删除。已关联用户：提示"删除后用户无相关权限"
API 接口
GET /api/v1/tenant/tree — 租户树
GET /api/v1/tenant/permissions — 租户功能权限
PUT /api/v1/tenant/permissions — 更新租户功能权限
GET /api/v1/assets — 资产列表（分页+筛选）
POST /api/v1/assets/sync — 同步资产数据
PUT /api/v1/assets/allocate — 资产划拨
POST /api/v1/users — 创建用户
GET /api/v1/users — 用户列表
PUT /api/v1/users/{id} — 编辑用户
DELETE /api/v1/users/{id} — 删除用户
POST /api/v1/users/{id}/reset-password — 重置密码
GET /api/v1/roles — 角色列表
POST /api/v1/roles — 创建角色
DELETE /api/v1/roles/{id} — 删除角色
测试要点
  • 租户层级增删改查 + 级联限制
  • 资产划拨后归属租户更新 + 记录生成
  • 两步创建用户（已存在/不存在邮箱分支）
  • 管理员角色保护（不可删除/不可编辑）
  • 数据权限隔离验证
04

租户管理

▼
模块概述

面向各层级租户管理员，管理下级租户的创建、编辑、开通管理员账号和删除。与业务管理中的租户信息不同，本模块是租户自管理视角。

业务逻辑
功能逻辑
新增租户企业名称（必填，唯一校验）、企业编码（必填，唯一校验）、企业地址/联系人/联系电话（选填）→ 在当前登录租户下新增下级
编辑租户企业编码不可修改。企业名称/地址/联系人/联系电话可修改
开通账号两步流程：Step1 填写昵称+邮箱（角色固定为管理员置灰）→ Step2 创建或关联用户中心账号
删除租户未开通管理员账号 → 可删除；已开通 → 不可删除
筛选企业编码（精准）、企业名称（模糊）、管理员账号（精准）、创建时间（日期区间）
列表企业编码 / 企业名称 / 企业地址 / 联系人 / 联系方式 / 管理员账号 / 创建时间
API 接口
GET /api/v1/tenants — 租户列表（分页+筛选）
POST /api/v1/tenants — 新增下级租户
PUT /api/v1/tenants/{id} — 编辑租户
DELETE /api/v1/tenants/{id} — 删除租户
POST /api/v1/tenants/{id}/setup-admin — 开通管理员账号
测试要点
  • 新增租户唯一性校验（企业编码+名称）
  • 删除租户限制（已开通不可删）
  • 两步开通账号流程（存在/不存在邮箱）
  • 开通后同步管理员角色 + 管理员用户

车辆管理

▼
模块概述

对当前租户及下级租户名下所有车辆的资产管理，包括车辆列表展示、批量导入更新、车辆详情查看（含7类台账记录）。

数据模型 — 车辆列表
字段说明
VIN码车辆唯一标识，17位字母数字
车牌号最多8字符
绑定设备IDOBD设备ID
车型车辆型号
外观车辆颜色
电池版本电池版本号
购车时间YYYY-MM-DD
车龄(当前日期-购车日期)/365，保留1位小数
总里程数累计行驶里程(km)，取整
最后位置逆地址解析，过长省略+气泡展示
批量导入校验逻辑
校验项处理方式
文件格式非 .xls/.xlsx/.csv整个文件导入失败，提示"请上传正确的 Excel 文件格式"
文件 > 50MB整个文件导入失败，提示"文件过大，请分批导入"
VIN码为空跳过错误数据，正确数据导入成功，生成失败明细
VIN码非17位跳过错误数据，正确数据导入成功，生成失败明细
VIN码重复（同一文件）仅保留首行
VIN不在当前租户资产池跳过，生成失败明细

可覆盖字段：外观、车牌号  |  不可覆盖：VIN码、购车时间、车型、类型、绑定设备号、车龄、总里程数、电池版本、最后位置

车辆详情 — 台账 Tabs
Tab核心字段数据来源
风控预警记录VIN、车牌号、预警名称（23种枚举）、预警内容、预警时间车身CAN信号
驾驶预警记录VIN、车牌号、预警名称（6种ADAS枚举）、预警内容、预警时间车身CAN信号

行程记录VIN、车牌号、开始/结束时间、起点/终点、行驶里程、行程时长、平均车速、预警次数OBD+车身
维修记录VIN、车牌号、维修类型、维修描述、开始/结束时间、记录人、维修状态手动创建/一键报修
里程报表折线图展示（X轴时间/Y轴里程），支持日/周/月/年维度切换里程数据聚合
UI 组件拆解
组件说明
车辆列表页筛选区（VIN/车牌号/设备ID/电池版本/车龄）+ 表格 + 批量导入按钮 + 分页

里程报表Tab维度切换（日/周/月/年）+ 折线图
API 接口

围栏管理

▼
模块概述

在租户维度管理电子围栏，支持创建中心点围栏和自定义围栏，配置使用车辆，控制围栏生效/失效。

业务逻辑
功能逻辑
围栏列表字段：序号/围栏名称/类型（中心点/自定义）/使用车辆数/状态（生效中/未生效）/地址/操作人/操作时间
新建围栏中心点：名称+地址+半径（≤100km）+预警类型（出栏/入栏）+地图选点。自定义：名称+地图标记点位（≤99个），点位需闭合
编辑围栏同新建逻辑
开关生效/未生效切换，关闭需二次确认
删除仅围栏关闭（未生效）时可删除，需二次确认
车辆配置点击"使用车辆"数字进入配置页，可添加/删除使用该围栏的车辆
筛选围栏名称（模糊）、类型（下拉单选）、状态（下拉单选）、操作时间（区间）
API 接口
GET /api/v1/fences — 围栏列表（分页+筛选）
POST /api/v1/fences — 创建围栏
PUT /api/v1/fences/{id} — 编辑围栏
DELETE /api/v1/fences/{id} — 删除围栏
PUT /api/v1/fences/{id}/toggle — 开关围栏
GET /api/v1/fences/{id}/vehicles — 围栏关联车辆
POST /api/v1/fences/{id}/vehicles — 添加关联车辆
DELETE /api/v1/fences/{id}/vehicles/{vin} — 移除关联车辆
测试要点
  • 中心点围栏半径校验（≤100km）
  • 自定义围栏点位闭合校验
  • 围栏开关状态流转
  • 删除限制（仅未生效可删）
  • 地图选点/标记交互
07

风控预警

▼
模块概述

集中展示和管理车辆运营中产生的各类风险报警信息，包括围栏出入、故障报警、电池报警。支持一键生成维修工单，打通维修管理流程。

功能结构
Tab类型数据来源核心操作
围栏报警基线复用OBD设备查看详情（含地图围栏范围）
故障报警定制OBD设备查看详情 + 一键报修
电池报警定制OBD设备查看详情 + 一键报修
围栏报警 — 字段
字段说明
报警类型出栏报警 入栏报警
围栏名称触发预警的围栏名称
报警位置触发时车辆所在位置
详情地图展示预警上报点位 + 围栏范围
故障报警 — 字段与逻辑
项目说明

系统管理

▼
模块概述

面向各层级租户用户的系统级管理，包括用户管理和角色管理。与业务管理中的用户/角色管理功能一致，但视角为租户用户侧。

用户管理
功能逻辑
用户列表根据登录用户所在租户层级查询该租户下用户。字段：序号/用户昵称/登录邮箱/角色/创建时间/操作
新增用户两步流程（同业务管理-用户管理）
编辑用户回显当前信息，管理员角色置灰不可修改
重置密码二次确认→重置为8位随机密码（大写字母+数字）→弹窗展示+复制
删除用户仅删除当前租户下的角色权限关系。管理员账号不可删
筛选用户昵称（模糊）/ 登录邮箱（模糊）/ 角色（多选）/ 创建时间（日期区间）
角色管理
功能逻辑
角色列表按创建时间正序展示。管理员角色由系统自动创建，不可编辑/删除
新增角色输入名称（唯一校验）→ 配置功能权限（范围=该租户层级可配置范围）→ 数据权限默认全部下级
删除角色未关联用户→二次确认后删除。已关联→提示"删除后用户无权限"
租户管理员账号同步租户创建→同步管理员角色→设置管理员账号（角色固定不可选）→同步管理员账号到用户管理

PUT /api/v1/sys/users/{id} — 编辑用户

POST /api/v1/sys/users/{id}/reset-password — 重置密码
