- **扁平化页面架构**：模块由多个独立的 React 函数组件（如 `Dashboard.tsx`, `Monitor.tsx`）构成，直接映射为应用的路由页面，无内部子模块嵌套。
- **状态驱动视图**：依赖全局 Zustand Store (`useAppStore`) 管理页面内的标签页切换状态（如 `_mt`, `_bt`, `_dt`），实现复杂页面（如监控、电池、驾驶）的子视图路由控制。
- **地图与图表集成**：深度集成 `react-leaflet` 用于车辆定位、轨迹回放及电子围栏绘制；使用 `react-chartjs-2` 渲染驾驶风险、能耗趋势等业务统计图表。
- **Mock 数据层解耦**：所有业务数据通过 `@/api/mock` 统一获取，实现了 UI 层与数据源的解耦，便于前端独立开发与演示。
