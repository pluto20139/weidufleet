- 全局状态管理采用 Zustand 的 persist 中间件，仅持久化用户身份、语言偏好及当前租户等关键会话数据，避免存储临时UI状态。
- API 客户端通过请求拦截器自动从 Store 获取 Token 并注入 Authorization 头，响应拦截器统一处理 401 未授权错误并触发登出流程。
- 国际化文案键名采用模块化前缀命名法（如 'menu.home', 'veh.plate'），并在 Sidebar 和 Topbar 等布局组件中通过 useTranslation Hook 动态渲染。
- 业务类型定义集中在 types/index.ts 中导出，涵盖车辆(Vehicle)、报警(AlertRecord)、围栏(FenceInfo)等核心实体，确保全链路类型安全。
